The Butcher is an HTML5, CSS3 and Javascript Template, totally responsive, using the acclamed Bootstrap 3.0 Framework.

The Butcher compatible with:

The Butcher Offers the possibility to Choose between 5 Different Theme Options:

  • Original Theme
  • Light
  • Turtledove
  • Darkbrown
  • Zephir

They Are simple CSS Files Inside each HTML Head.

Setting Up your Favourite Theme

Open file main.js in js folder and go to row 500:

Theme.setStyleSheet();

inside parentheses you just have to declare your favourite schema, choosen in template preview between those names:

  • light
  • turtledove
  • darkbrown
  • zephir

Here is a example of use:

Theme.setStyleSheet('light');

If you want to use the Default Theme, please leave parentheses blank.

How to Activate Menu Second Version (with Height Resizing)

We Experienced that some users needed a Menu version without resizing animation effet in Height, because they experienced a Logo Cut. We decided to put this feature as default to prevent this problem.

If you want to see how second version works, Open file index_menu_second_version.html

<body>

        <div class="wrapper animation">

            <!-- BEGIN TOP -->

            <div class="main-top" data-version="version2">

                    <!-- BEGIN TOP-BAR -->

                    <div class="top-bar">

data-version="version2" declaration is needed to swith menu to second version.

Animations

Animations are made using animation.css library, which can be consulted from this link:

 

http://daneden.github.io/animate.css/

Example of use:

<div class="headline-group">

                                    <i data-animation="fadeInUp fast" data-delay="0" class="headline-icon icon-about-us animated"></i>

                                    <h2 class="text-shadow-style" data-animation="fadeInUp fast" data-delay="0.2">Our Restaurant</h2>

                                    <h6 data-animation="fadeInUp fast" data-delay="0.5">Our Buffalo Meat is a Pleasure for your Senses! Call Us and choose your Seat</h6>

                                </div>

data-animation: takes 2 values: speed and animation

data-delay: receives takes a Value between 0 and 1

Use jquery.animsition.js library for animated page transitions, which can be consulted from this link:

http://blivesta.github.io/animsition/

 Example of use:

<div class="wrapper animsition">
        <a href="blog.html" class="animsition-link">Blog</a>
</div>

Parallax

Parallax uses  jquery.stellar.js. library.

If you want to use Parallax effect, this is the syntax:

 

<div class="parallax-wrapper">

                    <div class="parallax" data-stellar-background-ratio="0.5" data-stellar-vertical-offset="-500">

                        <div class="container">

                            <div class="row">.....

                …..

               </div>

</div>
